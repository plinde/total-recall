## Quick Download

Run the download script to fetch all images:

```bash
cd ~/workspace/github.com/plinde/total-recall/images
./DOWNLOAD.sh
```

This will download all referenced GIFs and images to the correct locations.

## Meme Collection

Additional Total Recall memes can be placed in the `total-recall-memes/` subdirectory.

Suggested themes:
- **Context Loss**: Quaid's confused face, memory wipe scenes
- **Context Restoration**: Memory recall sequences, "It's all coming back!"
- **Token Limits**: Eyes popping out, reactor overload
- **Successful /compact**: Kuato wisdom, "Open your mind"
- **Multiple Sessions**: The three-breasted woman ("when you have too many contexts")
